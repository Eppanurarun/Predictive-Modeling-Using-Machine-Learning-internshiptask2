# 🌸 Predictive Modeling Using Machine Learning – Iris Dataset

## 📊 Project Overview

This project focuses on building machine learning models to predict Iris flower species using the famous Iris dataset. The goal is to apply supervised learning techniques, evaluate model performance, and identify the best-performing classification model.

## 🎯 Objectives

- Build predictive models for flower species classification
- Apply supervised learning algorithms
- Train and test models for accuracy
- Evaluate model performance using confusion matrix and ROC curve
- Visualize important features and model results

## 🛠️ Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📁 Dataset Information

The Iris dataset contains 150 records of Iris flowers with the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species (Target Variable)

The target is to predict the species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

## 🔧 Machine Learning Algorithms Applied

### 1. Logistic Regression
Used for baseline classification and probability prediction.

### 2. Decision Tree Classifier
Used for rule-based classification and feature understanding.

### 3. Random Forest Classifier
Used for improved accuracy through ensemble learning.

## 📈 Model Evaluation

The following evaluation techniques were used:

- Accuracy Score
- Confusion Matrix
- Classification Report
- ROC Curve
- Feature Importance Visualization

## 📊 Key Findings

- Random Forest achieved the highest accuracy among all models
- Petal length and petal width were the most important features
- Iris-setosa was clearly separable from other species
- Logistic Regression required feature scaling for better convergence
- The dataset required minimal preprocessing due to its clean structure

## 📉 Visualizations Included

- Confusion Matrix Heatmap
- ROC Curve
- Feature Importance Bar Chart
- Accuracy Comparison Across Models

## 🚀 Conclusion

This project demonstrates practical implementation of supervised machine learning using classification algorithms. Random Forest performed best and successfully predicted flower species with high accuracy.

The project helped in understanding model training, testing, evaluation, and performance comparison using real-world machine learning workflows.
